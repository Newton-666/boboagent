# Web Design Standard v2

> keywords: web, 网页, 网站, blog, 博客, HTML, CSS, 前端, 页面, 设计, 布局, UI, landing, homepage
> Bobo 在做任何网页相关任务时必须遵守本文档。本文件是硬约束，不是建议。
>
> **设计哲学**: 不接受 AI 的默认输出。不接受默认紫色渐变、不接受 Emoji 装饰。
> 每个内容单元都是独立的入口，不是视觉装饰。

## 约束

### 绝对禁止（违反任一条 = 不合格，必须重新生成）

以下行为不是"不推荐"，是**被禁止的**。如果你的输出包含以下任何一项，说明你违反了本文档：

- ❌ Emoji（任何 Unicode 表情符号：😊 🎉 ✅ ❌ 🚀 💡 🔥 ⭐ 等）
- ❌ CSS gradient / linear-gradient / radial-gradient（任何渐变）
- ❌ 紫色系渐变及其变体
- ❌ `!important`
- ❌ 内联 style 属性
- ❌ 没有 alt 文本的 `<img>` 标签

### 必须遵守

- 语义化 HTML（`<article>` `<nav>` `<section>` `<header>` `<footer>` `<main>`）
- 所有按钮使用 `<button>`，不是 `<div onclick>`

## 内容架构

- 每一个卡片、链接、窗口必须是**独立可访问的内容单元**
- 卡片不只是视觉元素——它有自己的 URL 或可展开的独立状态
- 链接不是锚点跳转——是内容之间的导航关系
- 窗口/弹窗必须可以独立打开（不是悬浮依赖父页面的 fragile 状态）

## 配色

只能从以下 5 组中选择一组。格式：主色 / 文字色 / 背景色。

| # | 主色 | 文字色 | 背景色 | 风格 |
|---|------|--------|--------|------|
| 1 | `#FF8A00` | `#363636` | `#FFF2DF` | 橙色系 |
| 2 | `#0175CC` | `#B78D77` | `#FCF5E2` | 蓝色系 |
| 3 | `#5357A0` | `#FED81F` | `#FEFFEF` | 深蓝+黄 |
| 4 | `#0B7E9D` | `#D7B9A1` | `#F9F6E5` | 青色系 |
| 5 | `#1F4433` | `#FF8B31` | `#FFFBF0` | 深绿+橙 |

## 交互

- 所有按钮必须有 hover 状态的视觉反馈（颜色变化/阴影/缩放均可）
- 所有链接必须有 hover 和 visited 样式区分

## 审美

- 留白比内容多——不要把信息塞满屏幕
- 字体层级不超过三级（标题/正文/辅助）
- 颜色不超过页面中已定义的四色（主色/辅色/背景/强调）
- 不接受 AI 默认输出——每一处设计决策都应有理由

## 验收

- [ ] 移动端 375px 宽度不横向滚动
- [ ] 所有按钮可 hover
- [ ] 颜色来自上述 5 组之一
- [ ] 无渐变（除非用户明确要求）
- [ ] 无 Emoji
- [ ] 每个卡片/链接可独立访问
